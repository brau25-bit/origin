Diagrama Rest & Spread JS

Rest Parameters (junta argumentos en un array) 

function sumar(...numeros) {
  return numeros.reduce((a,b)=>a+b,0);
}
Spread Operator (expande arrays u objetos) 

const arr1 = [1,2];
const arr2 = [3,4];
const combinado = [...arr1,...arr2];
Combinando Rest + Spread 

function agregarAmigos(persona, ...amigos) {
  return {...persona, amigos};
}
